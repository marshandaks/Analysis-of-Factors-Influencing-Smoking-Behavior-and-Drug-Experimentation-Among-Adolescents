# Analysis-of-Factors-Influencing-Smoking-Behavior-and-Drug-Experimentation-Among-Adolescents

## Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis dan mengidentifikasi berbagai faktor yang memengaruhi perilaku merokok dan kecenderungan eksperimen obat-obatan terlarang di kalangan remaja. Analisis ini menggunakan data statistik untuk mengungkap hubungan sebab-akibat dan variabel-variabel penentu yang signifikan yang dapat menjadi dasar untuk program pencegahan dan intervensi yang lebih efektif.

## Tujuan

* Mengidentifikasi faktor-faktor sosio-demografis, psikologis, dan lingkungan yang berkontribusi pada perilaku merokok remaja.
* Menganalisis korelasi antara variabel-variabel tertentu dengan tingkat eksperimen obat-obatan terlarang di antara remaja.
* Menghasilkan model atau temuan statistik untuk memprediksi risiko perilaku merokok dan eksperimen obat-obatan.

## Metodologi

Analisis data dilakukan menggunakan pendekatan statistik multivariat, yang mungkin mencakup analisis regresi berganda (seperti yang ditunjukkan dalam metadata file proyek) untuk mengukur pengaruh relatif dari setiap faktor terhadap perilaku target.

## Alat yang Digunakan

| Alat | Kegunaan Utama dalam Proyek |
| :--- | :--- |
| *MiniTab* | Digunakan untuk analisis statistik dan pemodelan, termasuk regresi (berdasarkan jenis file .mpx yang diunggah). |
| *PowerBI* | Digunakan untuk visualisasi data interaktif dan pembuatan dashboard untuk menyajikan temuan analisis (berdasarkan jenis file .pbix yang diunggah). |
